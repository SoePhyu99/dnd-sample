React Drag and Drop Board (Kanban)

A simple Kanban-style board built with React, TypeScript, and react-beautiful-dnd.
Supports multiple columns and dragging cards between columns.

Features

- Multiple columns (To Do, In Progress, Review, Done)
- Drag and drop cards within and across columns
- Fully typed with TypeScript
- Built with React Hooks
- Clean, immutable state updates

Tech Stack

- React
- TypeScript
- react-beautiful-dnd

Note: react-beautiful-dnd is deprecated but still functional.
Recommended alternative: @hello-pangea/dnd
